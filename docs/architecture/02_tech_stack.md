# Tech Stack

The `smartplaylist` project utilizes the following technologies:

| Technology      | Description                               |
|-----------------|-------------------------------------------|
| Python 3.11+    | The core programming language.            |
| FastAPI          | For building the MCP Server API.          |
| Typer           | For creating the command-line interface.  |
| Pydantic & Pydantic-Settings | For data validation and settings management. |
| `uv`            | For dependency management and packaging.  |
| `pytest`        | For automated testing.                    |
| `ruff`          | For linting and code formatting.          |
| `beets`         | For music library management.             |
