# Getting Help

We're here to help! Whether you're a user or a contributor, this guide will help you find the right resources.

## Reporting Bugs

If you've found a bug, please open an issue on our [GitHub Issues](https://github.com/smartplaylist/smartplaylist/issues) page.

When filing a bug report, please include:

- A clear and descriptive title.
- Steps to reproduce the bug.
- What you expected to happen.
- What actually happened.
- Any relevant logs, screenshots, or error messages.

## Feature Requests

We welcome suggestions for enhancements! If you have an idea for a new feature or an improvement to an existing one, please open an issue on our [GitHub Issues](https://github.com/smartplaylist/smartplaylist/issues) page.

Please describe:

- The problem you're trying to solve.
- The solution you'd like to see.
- Any alternative solutions you've considered.

## Asking Questions

If you have a question about how to use the project, it's best to ask it on our [GitHub Issues](https://github.com/smartplaylist/smartplaylist/issues) page. This allows the community to benefit from the discussion.

## Contributing

If you'd like to contribute to the project, please read our [**Developer Guide**](./DEVELOPER-GUIDE.md) to learn how to get started. We appreciate your help!
